## 💡 使用技巧

### 1. 打印检查清单
```bash
# 打印 TODO_NEXT_STEPS.md 中的任务
grep -E "^- \[ \]" TODO_NEXT_STEPS.md
```

### 2. 追踪进度
在每个文档中，用 `[x]` 标记已完成的任务：
```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
```

### 3. 设置提醒
把关键日期添加到日历：
- 发布日期
- 内容发布日期
- 里程碑检查日期

### 4. 分享给团队
如果有团队成员，可以分享：
- `PROMOTION_CHECKLIST.md` - 给所有人看
- `LAUNCH_PLAN.md` - 给负责发布的人
- `DEMO_SCRIPT.md` - 给制作视频的人
- `CONTRIBUTING.md` - 给开发者看
